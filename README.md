# Terraform + Docker Project (Custom NGINX WebPage)

## Objective
Provision a Docker container with a custom HTML page using Terraform.

## Steps to Run
1. terraform init
2. terraform plan
3. terraform apply -auto-approve
4. docker ps
5. terraform destroy -auto-approve

## Output URL
http://localhost:8080
